 unityプロジェクトのビルド方法

	以下の手順でビルドできます。
	1.unity/IshikawaAlarm/Assets/Scenes/MainScene.unityを開きます
	2.File/Build Settingsを開き、PlatformをAndroidに変更します
	3.Build SystemをInternal(deprecated)に変更します
	4.Buildボタンを押すとビルドできます

====================================================================================================================
 注意事項

	Unityで動かす際には、
	AndroidDataManagerオブジェクトにアタッチされているAndroidDataManagerスクリプトの
	Is Connect Android_のチェックを外してください。
